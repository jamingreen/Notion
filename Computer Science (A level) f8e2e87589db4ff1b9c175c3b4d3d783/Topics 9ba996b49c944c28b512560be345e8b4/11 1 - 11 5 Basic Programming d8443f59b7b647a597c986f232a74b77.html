Integrated Development Environment (IDE)

A software used to develop programs

  • Features for debugging
    • Use of breakpoints
      • Check a program and its variable when it runs to a specific point
    • Use of stepping
      • Run line by line
      • Find the point where error occurs
    • Variable watch window
      • Allow user to keep track of the variables changes
  • Features for coding
    • Different colours (for variable, string, logical keywords)
    • Auto indentation
    • Comment
    • Auto completion of keywords
    • Line numbers

Basic elements of programming

  • Iteration (For, while)
  • Selection (if, else)
  • Sequence (the sequence of executing the code)

Subroutine

A subroutine is a set of instruction with name

  • Advantages
    • Large programs are broken down into subtasks that are easier to program and manage
    • Each modules can be individually tested
    • Modules can be reused several times in a program
    • Large programs are much easier to debug and maintain

Argument vs parameter

Parameter is the one in definition of the function

Argument is the variable when call the function

Global variable and Local variable

Local variable is the variable define in a subroutine (function) which can only be used by that function

Global variable is the variable define in the main program which can be used by other subroutine

Recursive Functions

Have a base case

Will call itself

Move towards the base case

  • Advantages
    • More elegant
    • Quicker to code
    • It is shorter in coding
    • Some program are naturally recursive (merge sort)
  • Disadvantages
    • Large memory complexity
    • Harder to trace
