Components

RAM (Random Access Memory)

  • A form of primary memory
  • Used to hold data and programs in used
  • Volatile (It will be lost when power is lost)

Arithmetic Logic Unit (ALU)

  • Performs arithmetic, logical and shift operation on data
  • Arithmetic: add, substract, multiply, divide
  • Logical: AND OR NOT XOR
  • Shift: move bit to the left or right in a register

Control Unit

Direct the operations of the CPU

  • Controlling and coordinating the activities of the CPU
  • Managing the flow of data between the CPU and other devices
  • Controls buses
  • Accepting the next instruction
  • Decoding instructions
  • Storing the result data back in the back in memory

Registers

  • Program counter
    • Hold the address of the next instruction to be executed
    • Value is sent to MAR
    • After sending to MAR, It is incremented or changed to the address in CIR if the operation is a jump
  • Accumulator
    • Temporary storage for data being processed or during calculation
    • Input and output in the processor, used as a buffer or a gateway
  • Memory address register
    • Contains the address of instruction sent from PC
    • Contains the address of data sent from CIR
    • hold the address of a location in the memory that is to be fetched from or store to
  • Memory data register
    • Contains the data or instruction accessed from the main memory
    • Temporarily stores data that has been read or data that needs to be written (stored)
    • The address is reference from MAR
    • It act as a buffer
  • Current Instruction Register
    • Holds the current instruction being executed, divided up into operand and opcode
  • Why use register over RAM?
    • Faster access (than RAM)
    • Allow fast access to data that is needed frequently and for a specific purpose

Buses

Buses are a set of parallel wires which connect two or more components inside the CPU.

System Bus

Buses consist of a series of connectors that transfer signals between internal components

Consist of 3 separate buses: Control Bus, Data Bus, Address Bus

Control bus

It is used to transmit control signal between internal and external components. It coordinates the use of the address bus and data bus and provides status information between system components.
  • Bi-directional

Control signals (by the bus)

  • Memory Read: data is read from specific location to be placed onto the data bus
  • Memory Write: data is written into address location using this bus
  • Interrupt request: shows that a device is requesting access to the CPU
  • Bus Request: Indicate that the device is requesting use of data bus
  • Bus Grant: indicate the CPU has granted access to the data bus
  • Clock: used to synchronise operation

Data bus

It is used for transporting data and instruction between components.
  • Bi-directional
  • The width = the data transfer per operation
  • It contain the data of the address
  • If the data bus is the same width as a computer word, data can be transfer to and from memory in a single operation (It might take more than one operation to transfer data)

Address bus

It is used to transmit the memory addresses specifying where data is to be sent to or retrieved from

The width of the address bus determines the maximum possible memory address of the system

Fetch-execute Cycle

The sequence of operations that are completed in order to execute an instruction

Fetch

  1. Address from PC is copied to MAR
    1. Data in MAR is passed to the Address bus
    1. Read signal is sent to control bus
    1. RAM copied data to data bus
  1. Instruction held at that address is copied to MDR by the data bus
  1. The PC is increased by one
  1. The value in MDR is copied to CIR

Decode

  1. The contents of CIR is split to operand and opcode
  1. Additional data, if required, is fetch from memory
  • The opcode specify the operation
  • Operand store either the address of the data, which is copied to MAR

    Or the actual data, which is copied to MDR

    Then it is copied to the accumulator

Execute

  1. The decoded instruction is executed
