Firewall

A hardware or software that controls (filter and blocks) access to and from a network

  • Only some ports are opened so that only certain traffic is allowed to pass through
  • It is prevent unauthorised access to a network

Packet filtering

Packets of data are inspected by the firewall

  • It examine the source IP, destination IP, the protocol and the port being requested
  • The packet can be rejected or dropped (Reject will alert the sender, drop will not)
  • Limit network access according to administrator rules and policies.

Proxy servers

It makes a web request on behalf of your own computer, hiding the true request IP address from the recipient

  • Enable anonymous surfing
  • Filer undesirable online content
  • Log user data with their requests
  • Provide cache of previously visited sites to speed access

Encryption

Encoding plaintext message so that it cannot decipher

It is method of keeping data secure when transmitting through the internet

Malicious Software

It annoys the user or damage their data

Virus

  • Infect other program or data files (by embedding itself in them)
  • Requires user to spread

Worms

  • A standalone program that does not require user to spread

Trojan

  • Software programs that masquerade as other application
  • Cannot self replicate
  • Often serve to open backdoor in your computer

Phishing

Using email to manipulative a victims into visiting fake website and give away personal information

Code quality

Improve code quality and monitor unauthorised access

  • Guarding against

  • Use strong password for login credentials
  • Two factor authentication

Buffer overflow

When a program accidentally writes data to a location too small to handle it

neighbouring space might be overwritten which the new data might be read as instructions

SQL injection

A user can enter a SQL commands via online database forms to change the processing
