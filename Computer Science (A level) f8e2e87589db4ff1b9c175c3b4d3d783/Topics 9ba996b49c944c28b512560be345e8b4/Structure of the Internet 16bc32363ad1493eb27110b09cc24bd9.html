Definitions

Internet

A public interconnection of computer networks allow data to be send globally to any connected device

World Wide Web

A collection of hyperlinked documents accessible via the internet

WWW vs internet

  • Internet is the connections between computers
  • Internet use ICP/IP communications protocols
  • The WWW is a system of hyperlinked documents
  • WWW is accessed using the Internet
  • WWW use the HTTP protocol to transfer documents

Backbone

A set of dedicated connections that connect regional networks, usually controlled by Internet Service Providers (ISPs)

IP address (Internet Protocol)

Set of four numbers identifying a resource on a network

URL (Uniform Resource Locator)

The combination of the protocol and fully qualified domain name of a resource

Used to specified the means of accessing a resource across a network and its location

  • Structure

Domain Names System (DNS)

Dedicated Computers with an index of domain names and the corresponding IP address

Domain names must be unique

Structure

When the IP address of a given domain is not known, it is referred to a related domain server that may know

  • Resolving IP address
    • Domain names used instead of IP address to reference a resource on a network
    • Local DNS serve contacted for IP address
    • If cannot supply one, send a request to another authoritative DNS
  • How do local DNS servers know when new top-level domains are added

    DNS names are hierarchical starting with the root servers

    These root servers know the location of all top-level domains

    As top level domains are added, the root server are updated

    If local DNS server doesn’t have a record of an top-level domain, it contacts a root server, so it can be informed directly about any domain names in existence

Internet registry

Organization that control the allocation of domain names and IP address

Local Area Networks (LANs)

two ore more computers connected together within a small geographical area

  • Another network: Wide Area Networks (WANs)

Network Topologies

Bus Network

An arrangement where nodes are connected in a daisy chain by a single central communications channel

  • All node is connected to a single backbone cable
  • Each node is passive
  • The end of the backbone is connected to a terminator or a computer to stop rebound the signals
  • Data is sent one in a time
  • One direction only
  • Advantages
    1. Inexpensive to set up
    1. Device can easily be added
    1. Good for small network
  • Disadvantages
    1. Main cable is a point of failure
    1. limited cable length
    1. Performance degrades with heavy use, owning to data “collision”
    1. no security
  • Diagram

Star Network

An arrangement where a central node or hub provide a common connect point for all device

The central node is often a switch

A switch sends each communication to computer it is specify for

  • Advantages
    1. Easy to isolate problems
    1. Good performance, due to each computer having its won dedicated connection
    1. More secure if a switch is used as data is sent only to the recipient
    • Easy to isolate problems, as each device has its own cable
    • Devices can easily be added as the network does not have to be shut down
  • disadvantages
    1. Can be expensive as more wires is required
    1. Central device is point of failure
