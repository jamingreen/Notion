Graph Structure

  • A collection of data nodes
  • Connection are set between nodes
  • Edges can be directional or bidirectional
  • It can be weighted or unweighted

A graph is a set of vertices or nodes that are connected to one another by edges or arcs

There is only one route for each node to each node

  • Difference and similarity between graph structure and tree structure
    GraphTree
    There is no node called rootThere is a unique node called root
    A loop or cycle can be formedNo loop or cycle can be formed
    One directionalBidirectional
    Both are collection of nodes and edges
    Both are non linear
    Both are dynamic data structure

*loop = looping to itself, cycle = a cycle that can go back to itself through other node

Delete node from binary tree

  1. Find the node to be deleted (15)
  1. Go to the left child node (13)
  1. Find the right most node (14) with no right child node
  1. Get the address of the node (14) and the left child node of (14) (If there is one)
  1. Replace the right to pointer from node 13 the left child node of (14)
  1. Delete the content in (15)
  1. Change the pointer of parent node of the node to be deleted (12) from (15) to (14)
