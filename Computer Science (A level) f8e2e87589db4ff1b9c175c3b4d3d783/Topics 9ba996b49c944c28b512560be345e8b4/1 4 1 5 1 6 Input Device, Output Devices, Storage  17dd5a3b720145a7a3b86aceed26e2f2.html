Definition

Input Device

It is a device that can be used to put data and information into a computer

Output Device

It is a device that can be used to send information from the computer

Input devices

  • Keyboard — Used to enter text into a computer
  • Webcam — A camera used by a computer to take photos and record videos
  • Magnetic Stripe Reader — Used to read data from magnetic stripes on cards like credit and debit cards
  • Barcode Reader — Used in supermarket checkouts and access control systems to read information from printed labels and cards

Output Devices

  • Speakers — Used to output sound from a computer
  • Printers — Used to output information from a computer onto paper
  • Projector — Used to project video content from a computer to screen

Input and Output Devices

  • Touch screen — output information in a visual way like a monitor + allow touch input

Storage

Optical

Optical devices are read from and written to by lasers

  • Binary Information is represented by whether the disc reflect or scatter the incident laser
    • The pit in CD scatter light while the flat reflect light

CDs (Compact Disc)

  • It is for storing small quantities of information
  • Commonly used to store audio information but can also store text and images
  • Disadvantages
    • Easily damaged by scratches
    • Limited storage capacities
    • Slow transfer speeds
  • CD-ROM — The information is written to during production (can only read from)
  • CD-R — Can only be written to once
  • CD-RW — Can be read and rewritten multiple times

DVDs (Digital Versatile Disc / Digital Video Disc)

It is a optical device which has higher storage than CDs

  • Store digital videos

Blu-Ray

An advancement over DVDs

  • Five times storage than DVDs
  • Store high-resolution films

Magnetic

It stores binary information represented using two magnetic state (polarised and unpolarised)

It is read by a read/write head

Hard Disk Drive (HDD)

  • High capacities (500 GB to 5 TB)
  • Work by rotating magnetic platters under read/write head on an actuating arm
  • Slower data transfer speed
  • Tend to become damaged by movement (brittle platter + moving parts)

Magnetic Tape

A tape of polarised and unpolarised material

Floppy Disks

A thin magnetic Disk

Flash

  • Fast and compact
  • Use silicon semiconductor to form logic gate NAND and NOR to store electric charge in two states (high or low — which is 1 and 0)
  • Flash memory can be erased and reprogrammed electronically and is non-volatile
  • More expensive than optical and magnetic

Solid State Drives (SSDs)

  • Light and portable
  • Resistant to damage from movement
  • High data transfer rate
  • High cost
  • Limited lifespan — Every time a page is written into, it requires higher voltage to rewritten

RAM vs ROM

Primary storage

RAM (Random Access Memory)

  • Fast main memory
  • High access speed
  • More expensive
  • Volatile
  • Temporary storage

ROM (Read Only Memory)

  • Cannot be modified
  • Non-volatile
  • Store fixed sequence of instructions (Computer startup (bootstrap) routine)

Virtual Memory

Temporarily transfer data from RAM to disk storage due to lack of storage in RAM

Virtual Storage

  • Storing information remotely so than everyone in the same system can access it
  • Example:
    • Cloud storage services
  • It is an abstraction of multiple drives
  • Convenient
  • Limited by network speed
  • High cost
