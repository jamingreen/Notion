    • Will not interevent in the event but will only report the values

Detection System

  • Pressure sensors (detect change in pressure)
  • Acoustic sensors (pick up accuse sounds)
  • Infra-red sensors (pick up movement or heat in rooms)
  • Gas
  • Light
  • Moisture
  • pH
  • Temperature

Monitoring System

  1. User keying in a PIN code
  1. Sensors constantly monitoring the rooms
  1. Data is converted into digital form using ADC and sent to a micro processor

Sensor Feedback

  • Sensor constantly taking readings
  • If the reading exceed the pre-set values, then the microprocessor send signal to warn the user
  • Each sensor will feed into an interface box so that the microprocessor can pin-point which sensor is triggered
  • Monitoring continues until the deactivate the system
