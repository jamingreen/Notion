Primary Storage / memory

RAM (Random Access Memory) (Primary storage)

  • Can be read from and written to
  • Access is fast
  • It will contain
    • The operating system
    • The software in use
    • The data which the software is using

ROM (Read Only Memory) (Secondary storage)

  • Can hold memory even under no power

Different Storage device

Hard disk drive (HDD) Magnetic drive

Magnetic hard drives work by magnetic patterns being read off platters that mechanically spin at high speeds

  • High capacity (5Gb to 5Tb
  • Relatively cheaper

Solid state drive (SSD) Flash drive

Flash hard drives use memory chips. These can have their contents erased and overwritten when an electrical charge is applied.

Electrically programmable non-volatile flash memory

  • Advantage
    • More durable (no moving part — can survive drops and extreme temperature and pressure
    • Faster than hard disk (higher data transfer speed)
    • Consume less power (no moving parts)
    • More portable
    • Quieter than hdd
  • Disadvantage
    • Less capacity than hard disk
    • More expensive than hard disk

USB flash drive

  • Can use password protection
  • Useful for transferring data between computers

CD, DVD, BluRay

  • Why they different capacities?
  • Read only
  • Recordable
  • Re-writable

CDs

  • CD-ROM (the content is save at the time of manufacture and cannot be changed after)
    • Software distribution
  • CD-R (Can be written to it once)
    • Copying software
  • CD-RW (Can be written, read and rewritten)
    • Short and mid term backup
  • DVD / BluRay
    • High capacity which can used to store films
