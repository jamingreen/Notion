  • Discuss whether magnetic storage or solid state storage would be most suitable, taking account, taking account into the advantages and disadvantages of both. (Server, client machine, backup)
    • knowledge

      Magnetic device include magnetic hard disk drive and magnetic tape and floppy disk

      • The binary information is represented by the 2 magnetic state of the disk.
      • It can be written to and read from using read and write hand

      • Solid state drive use semiconductor to form logic gate to store electric charge in 2 state, representing the binary information (1 and 0)
      • Can be rewritten using electricity

    • Application
      • Hard disk drive is used for storing software, database and other that requires a large amount of storage. It is relatively cheaper per storage than solid state drive.
      • Solid state drive is used for program files than requires faster access. Solid state drive have a higher data transfer rate than magnetic drives. It uses less power as there is no moving part. It is suitable for portable devices as it is lighter and not vulnerable to damage by movement.

      • Server will require quick access to programs, software and data
      • Client machine might be carried here and there
      • Backup server requires a large amount of storage and is not moved
    • Evaluation
      • Solid state drive will be most suitable for server and client machine as it consume less power and have a higher data transfer speed, allowing the server to have higher processor speed.
      • It is suitable for client machine as it might be moved while magnetic hard disk drive is vulnerable to damage by movement due to its moving parts and brittle platter
      • Magnetic hard disk drive is most suitable for backup server as large amount of data is likely to be stored. It is cheaper than solid state drive and thus can reduce the cost of storage. It is also seldom accessed so the power usage will not be a problem
